Introduction

FastLSH-UI is a web-based user interface of FastLSH Engine. FastLSH engine uses LSH algorithm to do similarity search on hugh high dimensional data in a Fast way. User can set parameters for the engine and observe the system status when the engine is running.

In addition to the FastLSH Engine, A high dimensional data visualizer(HDDV) is also provided.

Use the FastLSH Engine

To use the FastLSH Engine, click Run the Engine -> Run the Engine in the navigation bar on the left then follow the instructions.

Use the HDDV

To use the high dimensional data visualizer (HDDV), click "Data Presentation -> High Dimensional Visualization" in the navigation bar on the left then follow the instructions.
